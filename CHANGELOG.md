# Changelog

## [v1.3.1]
- 新增图片下载校验与白名单处理，增强代码健壮性

## [v1.3.0]
- 新增文件哈希校验，防止图片重复学习

## [v1.2.2]
### Added
- 支持设置学习表情的最大数量限制。
- 支持在表情数量达到上限时，开启随机替换分类下旧表情的功能。

### Fixed
- 优化预设的替换提示词，提高代码健壮性。

### Changed
- 强化提示词，增加文本替换模式的解析日志。
- 修正 `ai_learning.prompt` 的配置描述。
- 增加关于文本替换模式无法工作情况的提醒文档。

## [v1.1.2]
### Added
- 新增黑白名单处理功能，支持控制群聊和用户的插件触发。
- 支持表情包学习的概率控制。
- 初步实现图片解析能力，并支持表情自主学习。

### Changed
- 修改表情学习默认配置。
- 格式化输出日志。
- 修正部分文档格式。

## [v1.0.0]
### Added
- 初步完成人设替换方法（文本替换模式）。
- 完善并支持 AI 规划模式发送表情。
- 添加插件配置方式的说明。
- 插件基础配置初始化。
